What five things...

...do you need to start learning caligraphy?

Learn Calligraphy by Margaret Shepherd

With over fifty 5-star reviews, this book by Margaret Shepherd is recommended by novices and experts alike. Reviewers say it is “nearly perfect” and a fantastic starting place for all beginners.

Six-Nib Lettering Set

Nibs, huh? Yes, that’s what they’re called, and some fine nibs these are. Reviewers say that this set, made in the good ol’ USA, is very well made and is great for the price.

Super Black Ink

This ink is blacker than black and smoother than smooth. It’s made with pigment rather than dye, which means it won’t fade over time. Used with a proper set of nibs, you’ll be laying down professional calligraphy in no time.

Calligraphy Marker Set

If you don’t want to jump straight into the world of nibs and dipping ink, you can start with this set of calligraphy markers. The set comes with three different sized tips to allow for a variety of lettering techniques.

Ruled Practice Paper

This pad of practice paper contains fifty 9-inch by 12-inch sheets. Each sheet is lined with a practice rule and slanted grid which is perfect for practicing various styles of lettering. The paper is heavy enough to prevent ink from bleeding through, and reviewers say the quality is excellent.
